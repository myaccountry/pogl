# Pogl (Pascal opengl render)  
Program that can render and manipulate .obj files  
  
**1. Clone repository**  
```Shell
git clone https://github.com/myaccountry/pogl.git  
```
(linux install guide only, for windows construct yourselfs)  
  
**2. Compile program**  
```Shell
make all
```  
  
**Use it!**  
```Shell
./pogl --help
```
